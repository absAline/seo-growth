# AI SEO 工具提示词模板

## 1. 文章优化分析
```
你是一个 SEO 专家。请分析以下文章并给出优化建议：

文章标题：[标题]
目标关键词：[关键词]

1. 标题优化建议（3个方案）
2. 内容结构改进
3. 关键词密度分析
4. 缺失的 H2/H3 子主题
5. 内链建议
6. 元描述优化
```

## 2. 生成文章大纲
```
我的目标关键词是 "[关键词]"，目标读者是 [读者画像]。
请生成一篇 SEO 友好的文章大纲，要求：
1. 包含 H1-H4 标题层级
2. 每个标题下标注应包含的关键点
3. 自然融入以下长尾关键词：[长尾词1, 长尾词2]
4. 考虑搜索意图：用户搜索此关键词想解决什么问题
5. 文章预计 2000 字
```

## 3. 竞争对手分析
```
分析以下 3 篇排名靠前的文章（URL），它们的关键词是 [关键词]：

[URL1]
[URL2]
[URL3]

请分析：
1. 每篇文章的优缺点
2. 内容缺口（它们没覆盖到但用户关心的点）
3. 我的文章如何超越它们（内容深度/格式/独到见解）
4. 推荐的文章结构和长度
```

## 4. 生成推广文案
```
我写了一篇新文章：[标题]
文章链接：[URL]
核心观点：[3句话摘要]

请为以下平台生成推广文案（每个100-200字）：
1. 知乎 - 以回答形式推荐
2. 掘金 - 技术社区风格
3. 微信公众号 - 朋友圈分享文案
4. 小红书 - 笔记风格
5. Twitter/X - 英文短推
```

## 5. 内容重新利用 (Content Repurposing)
```
请帮我把这篇文章重新利用为多种格式：

原文：[粘贴内容]

生成：
1. Twitter/X 线程（5条推）
2. LinkedIn 帖子（200字）
3. 小红书图文笔记（标题+正文+标签）
4. 知乎回答版本（500字）
5. 邮件简报版本（150字）
6. 3张 Canva 配图描述
```

## 6. SEO 故障排查
```
我的网站遇到以下 SEO 问题，请帮助诊断：

问题描述：[如：文章不被收录 / 排名下降 / 流量骤降]

网站信息：
- 域名：[URL]
- 建站时间：[日期]
- 文章数量：[数字]
- 最近更改：[如有]

请给出：
1. 可能的原因（按可能性排序）
2. 检查清单（具体操作步骤）
3. 修复优先级和建议
```

## 7. 生成 FAQ 结构化数据
```
为主题 "[主题]" 生成 FAQPage 结构化数据，包含 5-8 个用户最常问的问题。

每个 FAQ 包含：
- 问题（搜索友好）
- 答案（50-100字，含关键词）
输出 JSON-LD 格式。
```

## 8. 数据分析解读
```
以下是我网站 [时间段] 的数据，请帮分析并给出建议：

Google Analytics 数据：
- 总浏览量：[数字]
- 独立访客：[数字]
- 平均停留时长：[数字]
- 跳出率：[百分比]
- 流量来源：直接[%] / 搜索[%] / 社交[%] / 外链[%]
- 热门页面：[列表]

请输出：
1. 数据反映的 3 个关键发现
2. 每个发现对应的优化建议
3. 下周应重点做什么
```
