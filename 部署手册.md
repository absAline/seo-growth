# 部署手册：AI辅助个人网站流量增长实战

本项目的产出是 SEO 策略文档，无需部署。以下是将策略应用到实际网站的方式。

## 方式一：参考文档直接应用

`seo-tools/` 目录中的6个 Markdown 文件即为完整策略模板：
- `technical-seo.md` — 技术 SEO 配置指南
- `keyword-strategy.md` — 关键词研究和内容日历
- `ai-seo-prompts.md` — AI 辅助 SEO 提示词
- `analytics-setup.md` — GA4 + GSC 配置
- `social-distribution.md` — 多平台分发策略
- `90-day-growth-plan.md` — 90天执行计划

直接在个人博客/网站中按文档步骤执行即可。

## 方式二：GitHub Pages 展示

如需将这些文档作为公开站点：

1. 在 GitHub 创建新仓库
2. 开启 GitHub Pages（Settings → Pages）
3. 选择 main 分支 /docs 目录
4. 将文档复制到 docs/ 目录
5. 访问 `https://你的用户名.github.io/仓库名`

## 方式三：Vercel 部署文档站点

如需更美观的展示：

1. Fork 或复制 seo-tools/ 到新项目
2. 创建简单 HTML 索引页
3. 部署到 Vercel（选择静态站点）

## 后台编辑器

```bash
cd admin
npm install
node server.js   # 端口 4325
```
