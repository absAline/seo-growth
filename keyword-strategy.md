### 关键词来源
- Google Search Console - 已有排名词
- 知乎/百度下拉 - 用户真实问题
- AnswerThePublic - 长尾问题
- Google 相关搜索 - 相关词扩展
- Claude/ChatGPT - AI 批量生成

### Claude Prompt 模板
```
你是一个 SEO 关键词研究专家。我的博客主题是 [主题]。
请生成 20 个长尾关键词，要求：
1. 搜索意图明确（信息型/导航型/交易型）
2. 竞争度低（适合新站）
3. 每月至少有搜索量
4. 按搜索意图分类
输出格式：关键词 | 搜索意图 | 推荐文章类型
```

## 文章 SEO 优化清单

发布前检查：
- [ ] 标题含目标关键词（前 60 字）
- [ ] 描述含目标关键词 + 吸引点击
- [ ] URL 含关键词（短横线分隔）
- [ ] H1 标签唯一且含关键词
- [ ] 文中 3-5 次自然出现关键词
- [ ] 有内链链接到旧文章（2-3 个）
- [ ] 有外链引用权威来源
- [ ] 图片添加 alt 文本
- [ ] 文章长度 > 1000 字
- [ ] 添加结构化数据 (JSON-LD)

## 内链策略

### 枢纽页 (Cornerstone) 模式
```
                    [首页]
                   /   |   \
              [分类A] [分类B] [分类C]
              /  |  \   |  \
          [文章] [文章] [文章] [文章]
```

规则：
1. 每篇文章至少链接 2 篇相关旧文
2. 重要页面获得最多内链
3. 锚文本使用关键词而非"点击这里"
4. 定期检查死链

## 外链获取策略

### 免费外链途径
- 知乎回答 (留博客链接)
- 掘金技术文章 (底部签名)
- GitHub README (项目文档)
- 客座博客 (Guest Post)
- 资源页提交 (Best XX Tools)
- HARO (记者求助)
- 友链交换 (相关性优先)
